# Calculadora Científica em Python

Este é um projeto de calculadora científica modular operada via terminal.

## Funcionalidades
- Operações básicas (+, -, *, /)
- Trigonometria (Sin, Cos, Tan)
- Logaritmos e Exponenciais
- Raízes e Potências
- Constantes (PI, E)

## Como executar
1. Certifique-se de ter o Python 3.x instalado.
2. Execute o comando: `python main.py`
